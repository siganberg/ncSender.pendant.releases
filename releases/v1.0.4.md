

## What's Changed

### 🔧 Improvements
- Switched to TinyUSB native USB for more reliable serial communication
- Disabled DTR-triggered reset to prevent unintended reboots when connecting
- Simplified dongle display for a cleaner status view

### 🐛 Bug Fixes
- Fixed ESP-NOW unpair not properly removing paired device
- Fixed potential data handling issues when dongle is not paired
