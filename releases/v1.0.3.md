## What's Changed

### ✨ New Features
- Added ESP-NOW wireless dongle support for cable-free pendant communication
- Added battery monitoring via AXP2101 PMIC with real-time battery status
- Added emergency power-off via hardware timer and task watchdog for safety
- Added Waveshare expansion board support with physical buttons, encoder, and power button
- Added USB serial communication as a reliable wired connection option
- Added USB OTA firmware updates for easy over-the-wire upgrades
- Added Control screen with job control and override sliders (feed, spindle, rapid)
- Added Configuration screen with Show G-Code and Accuracy Mode toggles
- Added MPG jogging with encoder acceleration strategies and closed-loop DRO tracking
- Added continuous jog mode for 0.1mm step size with tuned Z axis thresholds
- Added dynamic max feedrate scaling from ncSender settings
- Added automatic WiFi fallback with skip-scan reconnect and blinking WiFi icon
- Added online firmware flasher for browser-based flashing
- Added Info screen with firmware version and device details
- Added touch support with long press actions

### 🐛 Bug Fixes
- Fixed boot logo not showing after power-off/on cycle
- Fixed memory issues causing crashes on both WiFi and BLE connections
- Fixed slider not updating when override reset target is reached
- Fixed spindle override reset not updating slider position
- Fixed status widget reactivity and Door state handling
- Fixed long press triggering double-action on touch screens
- Fixed license not persisting after WiFi activation reboot
- Fixed boot screen content centering for different screen sizes
- Fixed partition table parsing and dynamic version display on boot

### 🔧 Improvements
- Changed ESP-NOW connection icon from charging bolt to WiFi symbol for clarity
- Prioritized ESP-NOW over USB when both connections are available
- Improved Z axis jogging with immediate single step per tick and lower continuous jog threshold
- Added Accuracy Mode (renamed from Brake on Stop) with refined jog strategy
- Added exponential smoothing to adaptive feedrate for smoother jog transitions
- Reduced touch long press duration from 1.5s to 1s for faster response
- Improved encoder accuracy with atomic GPIO reads
- Improved jog accumulator drain and brake logic
- Optimized Control screen and header update performance
- Switched boot screen to LVGL with embedded logos
- Compact serial protocol for efficient USB communication
- Multi-line commands sent as JSON over USB and ESP-NOW
- Persistent axis selection across screen changes
- Kept WiFi active when USB wired is connected
